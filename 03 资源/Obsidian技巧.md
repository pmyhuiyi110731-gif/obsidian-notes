# 🛠️ Obsidian技巧

> 快速上手指南

---

## ⌨️ 快捷键

| 功能 | 快捷键 |
|:---|:---|
| 新建笔记 | Ctrl+N |
| 快速切换 | Ctrl+O |
| 搜索 | Ctrl+Shift+F |
| 命令面板 | Ctrl+P |
| 图谱视图 | Ctrl+G |

---

## 🔗 双向链接

- 输入 `[[` 创建链接
- 链接不存在的笔记会自动创建
- 按住 Ctrl 点击链接打开

---

## 📁 文件组织

- **00 收集箱**: 临时存放，每天清理
- **01 项目**: 当前在做的事
- **02 领域**: 长期积累的知识
- **03 资源**: 工具、模板
- **04 复盘**: 周/月总结
- **05 日记**: 每日记录

---

## 🔌 推荐插件

1. **Dataview** - 数据查询
2. **Templater** - 模板自动化
3. **Git** - 版本同步
4. **Kanban** - 看板管理

---

## 💡 小技巧

- 用 `#` 创建标签
- 用 `- [ ]` 创建待办
- 用 `>` 创建引用
- 用 `` ` `` 创建行内代码
