## 🎯 核心定位

**Claude Code** 是 Anthropic 推出的AI编程助手，集成在终端中使用，可以：
- 理解代码库结构
- 执行重构任务
- 自动化开发工作流

---

## 🔧 核心功能

### 1. 代码理解与导航
- 快速理解大型代码库
- 智能代码搜索
- 跨文件分析

### 2. 代码生成与编辑
- 自然语言描述 → 代码实现
- 批量重构
- 自动生成测试

### 3. 自动化工作流
- 脚本编写
- 批量处理文件
- CI/CD 集成
